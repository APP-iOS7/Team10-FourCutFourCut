# FourCutFourCut! 📸

    좋아하는 사진들로만 골라서 본인만의 인생네컷을 만들어 보아요!
    카메라 앞에선 뚝딱이가 되기 쉬운 사람들 주목!
    카메라 앞에서 어색하게 나온 사진들로 기록해서 아쉬워하지 마세요.
    이젠 본인들이 좋아하는 사진들로 인생네컷을 만들어서 간직해요.😃
